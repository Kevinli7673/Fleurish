# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing any code.

**This project is deliberately on SDK 54, not the latest.** Do not "upgrade" it.
Expo Go for SDK 55/56/57 was never approved to the Apple App Store, so SDK 54 is the newest
version that runs on a physical iPhone without a paid Apple Developer account. See
`handoff.md` §12 before changing any Expo version.
